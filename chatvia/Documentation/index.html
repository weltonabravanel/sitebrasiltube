Introduction

Thank you for purchasing Chatvia - Laravel Responsive Chat App Template.

Chatvia is built with Bootstrap 4.5.0 and Laravel 8.*, SCSS with responsive with all devices and supported with Dark, Light modes. You can change mode very quickly by doing a couple of changes. We have used pushed for live chat application. It has many features like one to one chat, group chat, contact, send files, online users, read and unread new messages from users, authentication pages and many more.

If you are facing any issues then please contact us via support from the ThemeForest. You can also send email for pre-purchase query on themesbrand@gmail.com
